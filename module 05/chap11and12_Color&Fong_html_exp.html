Chapter 11 Takeaways

The take away from chapter 11 is that ther are FOUR different ways to apply color but the recommended ones to use are rgb (or rgba) and hex("#").

Be sure to get very comfortable with using r'gb(255, 255, 255) or r'gba(255, 255, 255, 1) and HEX (which is represented either like this: #a'aa OR #1a1a1a) as well as the properties of :

background-color (to change background color) color (to change font color)

These pseudo classes are awesome and really bring your links to life. They must be used in a specific order. The mnemonic was:

"Let's Visit Harry Again" (from Harry Potter). Here it is in context on an a tag:

(link, visited, hover, active)

a:link { color: deeppink; text-decoration: none; }

a:visited { color: black; }

a:hover { color: deeppink; text-decoration: underline; }

a:active { color: darkcyan; }
