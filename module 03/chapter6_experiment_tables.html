Let's explore TABLES

below we have a 3x3 table. it has 3 "tr" tag pairs (Table Row).
and in each "tr" section are 3 "td" tag pairs (so 3 columns of Table Data).

2A 2B 2C
3A 3B 3C

you can substitue "th" tags for "tr" tags, and you
will get column and row Table Headings. for sake of informing readers,
we can add the "Scope" attribure to our th's
